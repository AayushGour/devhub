# Studio Docs

One document per studio. Each covers: what it is, file tree, architecture, every component/hook/util, data flow diagrams, and how to contribute.

| Studio | Doc | What it does |
|--------|-----|-------------|
| Token Studio | [token-studio.md](token-studio.md) | Tokenise text with GPT/Qwen/Phi/Llama tokenizers; compare side-by-side |
| Crypto Studio | [crypto-studio.md](crypto-studio.md) | JWT, hashing, HMAC, AES cipher, Base64, secure token generation |
| JSON Studio | [json-studio.md](json-studio.md) | 6-mode JSON editor: tree, graph, diff, JSONPath, schema, type codegen |
| Markdown Studio | [markdown-studio.md](markdown-studio.md) | Monaco editor + live preview + 11 themes + PDF/HTML export |
| Diagram Studio | [diagram-studio.md](diagram-studio.md) | Mermaid editor with 16 templates, pan/zoom preview, SVG/PNG export |
| RAG Studio | [rag-studio.md](rag-studio.md) | Local RAG: upload docs, embed in-browser, chat with quantised LLM |
| Image Studio | [image-studio.md](image-studio.md) | Batch image converter (7 formats, custom BMP/ICO encoders, ZIP download) |
| SVG Studio | [svg-studio.md](svg-studio.md) | Raster → SVG via 4 engines (Potrace mono/colour, ImageTracer, embed) |
| Repo Explorer | [repo-explorer.md](repo-explorer.md) | GitHub repo fetch → dependency graph → per-file AI wiki → RAG chat |
